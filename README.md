# A little axe for game server programmers

 MMORPG를 포함하여 게임 서버를 만들고 게임을 서비스 하는 것이 즐거워야 한다. 
 게임을 만들기 쉽고 견고하며 서비스 하기 수월하도록 하는 도구를 만들고 싶었다. 
 이제 경험이 충분하므로 잘 생각해서 구현한다. 

## 문서화 

 한글과 영문을 섞어 쓰는 점에 대해 양해 하시길. 

 어떤 때는 영어로 사고하고 정리하기가 더 수월할 때가 있고 
 어떤 때는 한글을 쓰는 것이 더 즐겁고 쉬울 때가 있다. 

 한글의 장점이 다양한 언어를 쉽게 흡수 가능하다는 것이니 
 메세지 패싱이라 안 적고 message passing이라고 적는 차이로 보시길. 

## Features 

 - Message passing / clustering / subscription / continuation 
 - C++ core 
 - Physics / Navigation / Scene / Instance 
 - Database 
 - Remote debugging / monitoring 
 - Wanring: 
    - Microsoft Windows only 
    - 여러 플래폼 지원은 더 많은 노력이 필요하다.
