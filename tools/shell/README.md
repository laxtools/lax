# Shell 

 features: 

  - query based
  - session kept 

## Usage

### CLI: 

 $ shell.exe 

 shell> open ip:port 

 shell> login id password
 shell> logout

 shell> query json 
 shell> query file
 shell> run file 
 shell> spool file 
 shell> spool off 


### Runs commands from file. 

 $ shell.exe file
